# Founder Mode Playbook (Nova)

## Daily Cadence (Energy-Friendly)
- 08:00: KPI snapshot + blocker list
- 11:00: one critical shipping objective
- 15:00: reliability + QA checkpoint
- 19:00: publish/launch readiness check

## Weekly Cadence
- Mon: priorities + resource allocation
- Wed: product discovery review (idea kill/keep)
- Fri: creative quality + distribution review
- Sun: system hardening + backlog pruning

## Decision Gates
- Gate A (Creative): pass Brand QA + Hook QA
- Gate B (Business): pass monetization hypothesis
- Gate C (Reliability): pass launch checklist + rollback path
- Gate D (Compliance): pass policy checklist

## Missing "Head Hancho" Role
If founder load exceeds decision bandwidth, create:
- `chief_integrator`: coordinates all department heads, resolves cross-team conflicts, enforces execution rhythm.

This role is recommended now.
